# Slack Updates

This script sends a message to a Slack channel using the Slack API.

## Prerequisites

- Ruby
- Bundler

## Installation

1. Install dependencies from the root directory:
   ```bash
   bundle install
   ```

2. Create a `.env` file based on the `.env.sample` file and add your Slack API token:
   ```bash
   cp .env.sample .env
   ```

## Usage

Run the script:

```bash
ruby slackupdates.rb
```
