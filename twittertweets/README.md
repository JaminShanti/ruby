# Twitter Tweets

This script allows you to post tweets using the Twitter API.

## Prerequisites

- Ruby
- Bundler

## Installation

1. Install dependencies from the root directory:
   ```bash
   bundle install
   ```

2. Create a `.env` file based on the `.env.sample` file and add your Twitter API credentials:
   ```bash
   cp .env.sample .env
   ```

## Usage

Run the script:

```bash
ruby tweetDigitalGlobe.rb
```
